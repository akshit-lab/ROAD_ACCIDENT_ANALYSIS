# ROAD_ACCIDENT_ANALYSIS

## OVERVIEW
An interactive data analytics project built with Python, Pandas, Matplotlib, Seaborn, and Plotly to analyze and visualize road accident trends. The project processes accident records to uncover patterns related to accident severity, weather conditions, road types, vehicle categories, time of day, and geographical locations, providing meaningful insights through clear and interactive visualizations.

## FEATURES
Interactive dashboard for exploring accident data across multiple dimensions.
Analyze accident severity by location, road type, weather, and vehicle category.
Visualize accident trends using bar charts, line graphs, heatmaps, and geographical plots.
Identify high-risk areas and peak accident timings.
Data-driven insights to support road safety analysis and decision-making.
